# Гостевой дом «Мишаня» — сайт + бэкенд

## Структура

```
mishanya/
├── index.html          Главная
├── photos.html         Фотографии дома (галереи по разделам)
├── amenities.html      Удобства
├── prices.html         Полный прайс
├── rules.html          Правила проживания
├── guests.html         Фото гостей
├── booking.html        Бронирование (календарь + калькулятор)
├── contacts.html       Контакты
├── css/                Стили
├── js/                 Скрипты (pricing.js — расчёт цены)
├── img/                Папки под фото: interior, exterior, bath, guests
└── backend/            Ядро на C# (ASP.NET Core 8)
```

## Как посмотреть сайт прямо сейчас

Откройте `index.html` в браузере — или запустите локальный сервер,
чтобы корректно работали переходы:

```bash
cd mishanya
python3 -m http.server 8000
# откройте http://localhost:8000
```

Календарь и калькулятор работают сразу (с демо-занятостью).
Реальные занятые даты, оплата и Telegram включаются при запуске бэкенда.

## Куда класть фотографии

| Папка            | Что туда                                    |
|------------------|---------------------------------------------|
| img/exterior/    | фасад, двор, баня-бочка, чан, очаг, мангал  |
| img/interior/    | комнаты, кухня, гостиная, бильярд, аквариум |
| img/bath/        | бассейн, чан крупно, накрытый стол          |
| img/guests/      | фото из отзывов гостей                      |

Формат: WebP, ширина ~1600px, качество ~80 (так сайт грузится быстро —
это важно для позиций в поиске). После добавления фото замените заглушки
`.ph` в HTML на теги `<img src="img/...">`.

## Запуск бэкенда на C#

Требуется .NET 8 SDK (https://dotnet.microsoft.com/download).

```bash
cd backend
dotnet restore
dotnet run
# API поднимется на http://localhost:5000
```

Положите файлы фронта в `backend/wwwroot/`, чтобы сайт и API работали
с одного адреса (тогда /api/availability и /api/book доступны фронту).

### Что настроить в `backend/appsettings.json`

1. **Циан-календарь** — уже вписан ваш `.ics`, занятые даты подтянутся сами.
2. **Telegram** — создайте бота через @BotFather, вставьте `BotToken`.
   `ChatId` узнайте: напишите боту любое сообщение, откройте
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates` → возьмите `chat.id`.
3. **ЮKassa** — зарегистрируйте магазин на yookassa.ru, привяжите карту/счёт
   для выплат, вставьте `ShopId` и `SecretKey`. В ЛК укажите вебхук:
   `https://ваш-домен/api/payment/webhook`.
   Пока ключи пустые — работает демо-режим (без реальной оплаты).

## Как считается цена (единая формула фронта и бэкенда)

- Проживание = сумма по ночам: Пн–Чт 18000, Пт 20000, Сб 25000, Вс 20000 (за каждые сутки)
- Гости свыше 6: +1000 ₽ за гостя за каждые сутки (дети до 10 лет — бесплатно, не в лимите; старше 10 — как взрослый)
- Дрова 2000 ₽/сутки · Набор в чан 1500 ₽ разово · Веник 500 ₽/шт · Собака 1000 ₽ разово
- Предоплата = 10% от итога. Депозит 10000 ₽ — отдельно, возвратный.

**Важно:** при изменении тарифов правьте синхронно в двух местах —
`js/pricing.js` и `backend/Pricing.cs`. Бэкенд всегда пересчитывает цену
сам и не доверяет сумме с фронта.
